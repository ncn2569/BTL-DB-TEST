Cần node.js và npm để chạy được dự án này. 
cách chạy dự án:
node server.js rồi mở link 
trước khi chạy cần tạo file .env
example của file biến môi trường .env:
DB_USER= (tên user)
DB_PASSWORD= (password)
DB_SERVER= DESKTOP-RH42EGI\SQLEXPRESS (ví dụ tên server)
DB_DATABASE=BTL_DB (tên database)
DB_PORT=1433 (tên port)
DB_ENCRYPT=false
